## A. ランキング精度

### P-01: 市場（1番人気）に Top-1 が未達

| 項目 | 内容 |
|------|------|
| カテゴリ | ランキング |
| 深刻度 | **P0** |
| 状態 | 未解決（プロジェクト目標の核心） |

**問題の内容**  
純能力 LambdaRank（市場情報不使用）の予測1位的中率が、同一テスト集合における1番人気（単勝オッズ最小）の的中率を下回っている。内部合否基準（Top-1 > 30%）は満たすが、**市場そのものには未達**。

**数値根拠**

| 指標 | 値 | 出典 |
|------|-----|------|
| Model Top-1 | **30.24%** | `pure_rank/data/02_features/eval_results.json` |
| 1番人気 Top-1 | **32.90%** | `pure_rank/data/02_features/ev_results_with_favorite.json` |
| ギャップ | **−2.66pp** | 同上 |
| NDCG@3 / Spearman | 0.5359 / 0.5048 | `eval_results.json` |

**影響**  
「市場情報なしで1番人気を超える」というプロジェクト目標に対し、依然 2.66pp の差がある。EV ベッティングでは市場が既に能力を織り込んでいるため、このギャップは ROI 改善にも不利に作用する（P-15, P-21 と連動）。

**関連制約**  
Top-1 > 40% はリーク疑いで即停止（憲法）。市場超えは段階的改善が必要。

### P-02: 福島競馬場で Top-1 が大幅に低い

| 項目 | 内容 |
|------|------|
| カテゴリ | ランキング / 開催場 |
| 深刻度 | **P0** |
| 状態 | 未解決（v46 対策不合格） |

**問題の内容**  
福島（course_code=3）はテスト期間 312 レースで Top-1 **22.76%** と、全体（30.24%）から **−7.48pp**。テストレースの **6.5%** を占め、弱点の集中度が高い。

**数値根拠**

| セグメント | Top-1 | n |
|-----------|-------|---|
| 福島 全体 | 22.76% | 312 |
| 芝 | 25.14% | 175 |
| ダート | 18.25% | 126 |
| 馬場=良 | 21.96% | 296 |

出典: `pure_rank/data/02_features/course_weakness_v39_course_slim.json`, `error_analysis_v39_course_slim.json`

**構造的原因（調査済み）**

- 外れ 241 件のうち、実際の勝ち馬が逃げ・先行だった割合が高い（P-29）
- 予測1位の `hist_same_course_win_rate` NaN 率 **57.7%**（他場より高い）
- v46 `hist_small_course_pool_win_rate_ts` 追加 → **−0.10pp** で不合格

**影響**  
夏・春の東北開催（福島）で予想精度が体系的に低下。EV 層でも福島は VALID で ev_lift **−15.9pp**（条件フィルタ不採用）。
